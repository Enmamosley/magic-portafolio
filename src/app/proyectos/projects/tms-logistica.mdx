# Proyecto: TMS Logística - Sistema de Gestión de Transporte

## 🧩 Contexto
Empresa de transporte de carga que operaba con procesos fragmentados en hojas de cálculo y sistemas desconectados para controlar embarques, facturación y cuentas por cobrar/pagar. Necesitaban una plataforma centralizada que integrara todas sus operaciones logísticas y cumpliera con la normativa fiscal mexicana (CFDI 4.0 con Carta Porte).

## 🎯 Objetivo
Desarrollar un Transport Management System (TMS) completo que centralice la gestión de embarques, automatice la facturación electrónica con complemento Carta Porte 3.1 y proporcione control total sobre las finanzas operativas de la empresa.

## 👥 Alcance del Proyecto
- 35+ modelos Eloquent con 25+ tablas en PostgreSQL
- 80+ componentes Livewire Volt
- Facturación CFDI 4.0 con complemento Carta Porte 3.1 vía Facturapi
- Complementos de pago y notas de crédito
- Cuentas por cobrar y cuentas por pagar
- DTOs para Carta Porte y Bill of Lading (BOL)
- Servicios de reportes y notificaciones por email
- ~50,000+ líneas de código

## 🔧 Tareas Realizadas
- Arquitectura y desarrollo del sistema completo con Laravel 11.x
- Implementación de 80+ componentes reactivos con Livewire 3 (Volt)
- Modelado de base de datos PostgreSQL con 25+ tablas y relaciones
- Integración con API de Facturapi para timbrado de CFDI y Carta Porte
- Desarrollo de módulos de cuentas por cobrar/pagar con conciliación
- Implementación de DTOs tipados para Carta Porte y BOL
- Diseño de interfaz con Tailwind CSS 3 y DaisyUI
- Sistema de reportes y notificaciones automatizadas por email
- Integración con catálogos SAT México

## 📈 Resultados Obtenidos
- Automatización completa del ciclo de embarque a facturación
- Cumplimiento normativo SAT con Carta Porte 3.1 sin intervención manual
- Visibilidad en tiempo real de cuentas por cobrar y pagar
- Reducción drástica de tiempos administrativos
- Plataforma escalable con arquitectura robusta
