# Proyecto: Inventory Plus - Gestión Avanzada de Inventario para Bagisto

## 🧩 Contexto
Las tiendas en línea que operan con múltiples almacenes necesitan herramientas avanzadas de control de inventario que Bagisto no ofrece de forma nativa: trazabilidad de movimientos, transferencias entre ubicaciones, gestión de códigos de barras y alertas automatizadas de stock bajo.

## 🎯 Objetivo
Crear un módulo open-source que extienda las capacidades de inventario de Bagisto 2.x, proporcionando a los comerciantes herramientas profesionales para el control total de sus existencias.

## 👥 Alcance del Proyecto
- Log completo de movimientos de inventario (audit trail)
- Transferencias entre almacenes con aprobación
- Generación y lectura de códigos de barras (EAN-13, UPC-A, Code 128, etc.)
- Alertas de stock bajo y crítico con notificación por email
- Importación y exportación masiva vía CSV

## 🔧 Tareas Realizadas
- Desarrollo del módulo en PHP 8.2+ para Bagisto 2.x
- Implementación de sistema de movimientos con trazabilidad completa
- Integración de librería picqer/php-barcode-generator para múltiples formatos
- Desarrollo de sistema de alertas con umbrales personalizables
- Motor de importación/exportación CSV para operaciones masivas
- Documentación técnica y publicación en GitHub

## 📈 Resultados Obtenidos
- Control total de inventario con historial de cada movimiento
- Operaciones entre almacenes gestionadas digitalmente
- Reducción de faltantes gracias a alertas automatizadas
- Módulo publicado como open-source para la comunidad Bagisto
