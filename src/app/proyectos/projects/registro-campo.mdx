# Proyecto: Registro de Campo - Aplicación Web

## 🧩 Contexto
Organización que necesitaba digitalizar sus procesos de registro en campo, reemplazando formularios en papel y hojas de cálculo por un sistema web accesible desde cualquier dispositivo.

## 🎯 Objetivo
Desarrollar una aplicación web robusta que permita registrar, consultar y gestionar datos recopilados en campo de forma eficiente y en tiempo real.

## 👥 Alcance del Proyecto
- Aplicación web responsive accesible desde móvil y escritorio
- Sistema de registro con formularios dinámicos
- Panel de administración con reportes
- Gestión de usuarios y permisos

## 🔧 Tareas Realizadas
- Desarrollo backend con Laravel 11
- Implementación de formularios dinámicos con validación
- Diseño de panel de administración con Blade y Tailwind CSS
- Sistema de autenticación y roles
- Despliegue en entorno productivo

## 📈 Resultados Obtenidos
- Digitalización completa del proceso de captura en campo
- Acceso en tiempo real a datos desde cualquier dispositivo
- Eliminación de errores por captura manual
- Mayor eficiencia en el procesamiento de información
