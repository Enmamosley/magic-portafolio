# Proyecto: Pabellón M - Sitio Web con Animaciones Avanzadas

## 🧩 Contexto
Pabellón M es un desarrollo comercial en Monterrey, Nuevo León. Su sitio original fue construido en Webflow. El reto era replicar el sitio completo — incluyendo sus animaciones de scroll, directorio de negocios y mapa interactivo — utilizando tecnologías modernas de código abierto.

## 🎯 Objetivo
Desarrollar una réplica pixel-perfect del sitio pabellonm.com con Astro 6 y animaciones avanzadas, demostrando que se puede lograr la misma calidad visual y de interacción que Webflow con código puro.

## 👥 Alcance del Proyecto
- 7 páginas: Home, Nosotros, Directorio (30 negocios), Mapa interactivo (9 pisos), Informes, Contacto y Aviso de Privacidad
- Animaciones de scroll con GSAP ScrollTrigger y SplitType
- Datos extraídos del sitio Webflow original
- Deploy automático con GitHub Actions a GitHub Pages

## 🔧 Tareas Realizadas
- Scraping del sitio original para extraer contenido, imágenes y estructura
- Desarrollo con Astro 6 y Tailwind CSS 4
- Implementación de animaciones con GSAP ScrollTrigger (word reveal, parallax cards)
- Creación de mapa interactivo con navegación por pisos
- Directorio con fichas de 30 negocios y filtrado
- Configuración de CI/CD con GitHub Actions para deploy automático
- Optimización de rendimiento y assets

## 📈 Resultados Obtenidos
- Réplica fiel del sitio original con tecnología moderna
- Animaciones de scroll fluidas y performantes
- Deploy automático en cada push a producción
- Demostración de viabilidad de migración Webflow → Astro
