# Proyecto: TEDx Jardines de Pereda - Evento de Ideas Innovadoras

## 🧩 Contexto
TEDx Jardines de Pereda es un evento independiente organizado bajo licencia TED, enfocado en crear un espacio donde ideas revolucionarias y voces inspiradoras se encuentren para desafiar los límites del pensamiento. El evento requería una presencia digital profesional que reflejara los valores de innovación y excelencia de la marca TED.

## 🎯 Objetivo
Desarrollar un sitio web moderno y funcional que facilite el registro de asistentes, presente información del evento de manera atractiva y genere expectativa en torno a las conferencias y ponentes, manteniendo la identidad visual característica de TEDx.

## 👥 Alcance del Proyecto
- Sitio web del evento con diseño TEDx oficial
- Sistema de registro de asistentes
- Integración con pasarela de pagos Stripe
- Newsletter y sistema de comunicaciones
- Optimización para conversión de registros

## 🔧 Tareas Realizadas
- Desarrollo con WordPress y tema personalizado
- Implementación de identidad visual TEDx oficial
- Integración con Stripe para venta de entradas
- Configuración de sistema de newsletter integrado
- Optimización para carga rápida y conversión

## 📈 Resultados Obtenidos
- Diseño que cumple 100% con estándares oficiales TEDx
- Sistema de venta de entradas funcional y seguro
- Experiencia de usuario optimizada para conversión
- Presencia digital profesional que genera confianza
