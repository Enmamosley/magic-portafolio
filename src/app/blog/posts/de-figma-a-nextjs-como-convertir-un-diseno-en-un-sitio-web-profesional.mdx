## Introducción

Figma es una herramienta increíble para diseñar interfaces, pero ¿cómo pasamos de un diseño bonito a un sitio funcional en Next.js? Aquí te explico el proceso paso a paso.

## Paso 1: Exporta tu diseño

Desde Figma, puedes exportar imágenes, íconos y obtener medidas precisas usando la herramienta de inspección.

## Paso 2: Estructura tu proyecto

En Next.js, crea una estructura básica:

```
src/
  components/
  pages/
  styles/
```

## Paso 3: Crea componentes reutilizables

Convierte cada sección de tu diseño (hero, testimonials, footer) en componentes separados. Usa TailwindCSS o CSS modules para mantener estilos organizados.

## Paso 4: Implementa interactividad

Agrega animaciones con `framer-motion`, enlaces dinámicos con `next/link`, y cualquier lógica que haga tu sitio más interactivo.

## Paso 5: Publica en Vercel

Next.js y Vercel trabajan perfectamente juntos. Haz push a GitHub y conecta el repo en Vercel para desplegar automáticamente.

## Conclusión

Convertir tus diseños de Figma a código puede parecer desafiante, pero con práctica y una buena estructura, lograrás sitios hermosos y rápidos.
